### …or create a new repository on the command line

echo "# Li-Jennifer.github.io" >> README.md

git remote add origin git@github.com:Li-Jennifer/Li-Jennifer.github.io.git

git remote add origin git@github.com:Li-Jennifer/Li-Jennifer.github.io.git
git branch -M main
